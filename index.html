Outline - GQRX Over TCP

Aug 20th, 2013

GQRX Client: Gqrx 2.2 – [http://gqrx.dk/download]

GQRX Server: Software:

Get:
  libusb-1.0.0-dev
  git clone git://git.osmocom.org/rtl-sdr.git
  cmake
 Do:
  cd rtl-sdr/
  mkdir build
  cd build
  cmake ../ -DINSTALL_UDEV_RULES=ON # On to allow privlages for non-root users
  make
  sudo make install
  sudo ldconfig
 Use:
   rtl_tcp -a 0.0.0.0
